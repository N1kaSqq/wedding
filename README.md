# Wedding Website - 20.06.2026

Одностраничный сайт-приглашение на свадьбу с мобильно-ориентированным дизайном.

## Структура проекта

```
wedding/website/
├── index.html          # Основная HTML страница
├── styles.css          # Все стили
├── script.js           # JavaScript функциональность
├── assets/             # Папка для изображений
│   ├── bow.png
│   ├── dresscode-girls-1.jpg
│   ├── dresscode-girls-2.jpg
│   └── dresscode-men.jpg
└── README.md          # Этот файл
```

## Разделы сайта

1. **Hero** - Заголовок с датой свадьбы и приветствием
2. **Локация** - Адрес места проведения (Усадьба Royal Hall)
3. **Тайминг** - Расписание дня (фуршет, церемония, банкет, финал)
4. **Дресс-код** - Цветовая палитра и примеры нарядов для гостей
5. **Детали** - Информация про подарки и пожелания
6. **Анкета** - Форма для RSVP с интеграцией Formspree.io
7. **Footer** - Прощание

## Следующие шаги

### 1. Добавить изображения

Экспортируйте из Figma и поместите в папку `assets/`:
- `bow.png` - Розовый бант (разделитель)
- `dresscode-girls-1.jpg` - Пример дресс-кода для девушек
- `dresscode-girls-2.jpg` - Пример дресс-кода для девушек
- `dresscode-men.jpg` - Пример дресс-кода для мужчин

См. инструкции в `assets/.gitkeep`

### 2. Подключить шрифт

Текущий placeholder: `'Brush Script MT', cursive`

Необходимо:
1. Получить название шрифта из Figma или ссылку на него
2. Подключить через Google Fonts или локально
3. Обновить CSS переменную `--font-script` в `styles.css` (строка 26)
4. Обновить `<head>` в `index.html` (добавить `<link>` для шрифта)

### 3. Настроить Formspree.io

1. Зарегистрируйтесь на [Formspree.io](https://formspree.io/)
2. Создайте новую форму
3. Получите endpoint URL (формат: `https://formspree.io/f/YOUR_FORM_ID`)
4. Замените в `index.html` (строка 241):
   ```html
   <form id="wedding-form" class="wedding-form" action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
5. Настройте email, на который будут приходить уведомления

## Функциональность

### Навигация
- Фиксированное меню в верхней части страницы
- Плавная прокрутка (smooth scroll) к секциям
- Подсветка активного пункта меню при скролле

### Форма
- Валидация обязательных полей
- Визуальная обратная связь при ошибках
- Сообщение об успешной отправке
- Интеграция с Formspree.io

### Адаптивность
- Mobile-first дизайн
- Адаптация для планшетов и десктопов
- Контейнер для контента на больших экранах

## Деплой

Сайт можно разместить на:

### GitHub Pages (бесплатно)
1. Создайте репозиторий на GitHub
2. Загрузите все файлы
3. Включите GitHub Pages в настройках репозитория
4. Сайт будет доступен по адресу: `https://username.github.io/repo-name`

### Netlify (бесплатно)
1. Создайте аккаунт на [Netlify](https://netlify.com)
2. Перетащите папку с проектом в Netlify Drop
3. Получите ссылку на сайт

### Vercel (бесплатно)
1. Создайте аккаунт на [Vercel](https://vercel.com)
2. Импортируйте проект из GitHub или загрузите файлы
3. Получите ссылку на сайт
